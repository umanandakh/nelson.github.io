Introduction

In this chapter, we study the Laplace transformation, which is one of several important transforms used in linear system analysis. The function of a transform tool is to convert operation of one type into operations of different type. The Laplace transform offers significant advantages. Through this tool, we model continuous time physical system with linear differential equations with constant coefficients. The Laplace transform has no physical significance except that it transforms the time domain signal to a complex frequency domain.

It is useful to simplify the mathematical comoputations and it can be used for the easy analysis of signals and systems. We use the Laplace transform to solve differential equations, the solutions are functions of the Laplace transformation variable ’s’ rather of the time variable ’t’. Therefore we must have a method for converting from function ’s’ back to function of t, this procedure is called Inverse Laplace transformation.

Definition

If the Laplace transform of a function F(t) is f(s), i.e, LF(t) = f(s) then F(t) is called an Inverse Laplace transformation of f(s). We also write F(t) = L−1f(s)
